# 🚀 Task Manager Web App

A simple yet powerful **Task Manager** web application built using modern web technologies. This project allows users to create, view, and expand tasks dynamically in real-time.

---

## 📌 Features

- ✨ Add tasks with title and description  
- 📄 View tasks instantly on the page (live update)  
- 🔍 "Read More" functionality to expand full task details  
- 🎨 Clean and responsive UI using Tailwind CSS  
- ⚡ Fast server-side rendering using EJS  
- 🌐 Backend powered by Node.js and Express  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Templating Engine:** EJS  
- **Other Concepts:** Routing, Dynamic Rendering, Form Handling
